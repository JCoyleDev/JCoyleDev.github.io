How To Create Your First GitHub Web Page

Preparation:

Before you create your web page in GitHub, there are a few things that will need to be set up beforehand.

  • You will need to install a program where you can create an HTML file and write the code for your web page. While you can technically use Microsoft's Notepad for this, the best practice is to use an industry standard program such as Visual Studio Code. Download VSCode by clicking here.
  • Create a new root folder where you will store the files for your repository. You can set this up anywhere you want to.
  • Choose an image that you would like to use as your profile image and save it to the new root folder that you created in the last step.
  • (Optional) Create a README file within your root folder that will tell others a bit about your web page.

Creating the HTML File:

Next you will want to create the HTML file which will provide the structure for your personal web page.

  1. In your chosen code editor, create a new file called "index.html" sans quotation marks.
  2. If you have some familiarity already with HTML, then go ahead and set up the document however you'd like. If you are completely new, I would recommend starting with a basic structure like this:
  3. In the body, you'll want to create a heading for your page using a header tag, such as h1 or h2. It can be as simple as "Your Name's Portfolio".
  4. Next, you'll insert your profile image below your heading using the img tag. Your src property for this tag will depend on where inside the root folder, you saved the profile picture. For this example, we will just assume you saved everything in a single folder.

Creating an Account:

Now that you have your HTML document created and saved in your root folder, it's time to create your GitHub account.

  1. Start by navigating to the GitHub homepage at https://github.com/.
  2. Click the "Sign Up" button in the upper right-hand corner of the page.
  3. Enter the email address you would like to link to your account, and choose a strong password.
  4. Pick a unique username that others can use to identify you on the site.
  5. Complete the remaining prompts to finish setting up your account. When you are done, you should be redirected to the home page. If you are not logged in, click login in the upper right-hand corner and type the login information you just created.

Starting a New Repository:

You're almost there! Now we are going to create the repository where you will upload the files for your web page.

  1. From the homepage, click the drop-down icon in the upper right-hand corner. On the drop-down menu, click "Your repositories".
  2. You won't have any repositories showing up yet, so next you'll click the green "New" button on the right-hand side of the screen.
  3. Title your repository whatever you'd like. You can even just retype your username. When you are done choosing the settings for the repository, click "Create repository" at the bottom of the form.
  4. On the following page, click the link that says "Upload files". You will be taken to a screen with a drag and drop uploader. Open a file explorer window on your desktop and navigate to the root folder you created that should now hold your index.html file and your profile photo. Drag and drop the contents of your root folder to the uploader in the browser. Click "Commit Changes".

Checking for Functionality:

Lastly, you should make sure that your webpage is live. If you type repositoryname.github.io in your address bar, your custom webpage should be visible. Congratulations! You've just set up your first personal web page using GitHub!
